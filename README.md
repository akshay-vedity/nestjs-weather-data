# nestjs-weather-data

## Description

Simple [Nest](https://github.com/nestjs/nest) Application that takes latitude and longitude and date, and returns the weather description for that location for a given date.

## Installation

```bash
$ npm install
```

## ENV

Rename .env_demo to .env

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
